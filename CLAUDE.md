# Claude Diagnostics

This is a cross-platform system diagnostic workspace.

Use `/start-diagnostic` to activate diagnostic expert mode.

## Key Files

| File | Purpose |
|---|---|
| `AGENTS.md` | Diagnostic persona, constraints, and full operational rules |
| `SYSTEM_PROFILE.md` | Machine hardware/software profile (generated by `scripts/discover.py`) |
| `ISSUES.md` | Issue tracker — active, known, resolved, no-solution |
| `sessions/journal.md` | Session history — findings, decisions, lessons |

## Scripts

| Script | Usage | Output |
|---|---|---|
| `scripts/discover.py` | `python scripts/discover.py` | `SYSTEM_PROFILE.md` |
| `scripts/health.py` | `python scripts/health.py` | `output/health-dump.txt` |
| `scripts/elevated.py` | `sudo python scripts/elevated.py` | `output/elevated-dump.txt` |

## Rules

- All diagnostic behavior is defined in `AGENTS.md` — follow it.
- This workspace supports Linux, macOS, and Windows.
- Never modify system files without explicit user approval.
